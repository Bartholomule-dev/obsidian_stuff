# Tournaments

Combat gauntlets where demons fight through multiple rounds for XP and stat rewards.

## Available Tournaments

### Pit Fighter's Trial (Tier 1)
Entry-level combat gauntlet for new demons.

| Property | Value |
|----------|-------|
| Level Req | 1 |
| Rounds | 3 |
| Entry Cost | Low |
| Rewards | XP, +2 Wrath, +1 Pride |

### Arena of Wrath (Tier 2)
Intermediate combat challenges.

| Property | Value |
|----------|-------|
| Level Req | 3 |
| Rounds | 4 |
| Entry Cost | Medium |
| Rewards | XP, +3 Wrath, +2 Pride, +1 Envy |

### Infernal Colosseum (Tier 3)
The ultimate combat proving ground.

| Property | Value |
|----------|-------|
| Level Req | 5 |
| Rounds | 5 |
| Entry Cost | High |
| Rewards | XP, +4 Wrath, +4 Pride, +2 Envy |

## Tournament Flow

1. **Entry**: Pay soul essence, check requirements
2. **Rounds**: Fight through consecutive battles
3. **Progressive Rewards**: Rewards scale with rounds won
4. **Completion**: Full rewards if all rounds won

## Strategic Value

Tournaments are the most efficient way to gain:
- **Wrath**: Key for Spike verbs (Intimidate)
- **Pride**: Key for Intel verbs (Read Soul)
- **Envy**: Key for Control verbs (Honeyed Words)

Combined with training, tournaments accelerate unlocking high-tier [[Verbs]].

## Requirements

- Demon must be slotted in roster
- Must meet level requirement
- Must have sufficient energy
- Soul essence entry fee
